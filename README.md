# GillonMcBotlin
An all-rounder JS bot for the /r/AFL Discord Server.

You'll need a config.json file in the main directory, which should look like this:
```json
{
    "client_id": "CLIENT",
    "token": "TOKEN",
    "prefix": "+",
    "owner": "159609273058459648"
}
```

To install, do
```
npm install
```
assuming you have Node.js installed
